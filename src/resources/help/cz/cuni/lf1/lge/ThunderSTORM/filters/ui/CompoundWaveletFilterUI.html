Wavelet filter

ThunderSTORM uses a wavelet filter based on the method of [1]. This is our filter of choice in ThunderSTORM. The filter is an undecimated wavelet transform with a kernel determined by a third-order B-spline. We use three wavelet planes, each corresponding to a range of different spatial frequencies:

  1. the first plane is given by k1 = T and contains mainly the high spatial frequencies present in the input image including noise;
  2. the second plane is given by k2 = T and contains structures with sizes close to the diffraction limit;
  3. the third plane is given by k3 = T and contains the lower frequency image components.

Each of the planes can be used for different operations. For example we use the first plane for threshold estimation for the subsequent molecular detection step. We use the second plane (or if selected by the user, the third plane) as an input image for the detection of molecules.
